# Quiz App 🎯

A simple and interactive Quiz Application built using HTML, CSS, and JavaScript.
This app allows users to answer multiple-choice questions and displays the final score at the end.

---

## 🚀 Features

* Multiple-choice questions
* Instant feedback (correct / incorrect answers)
* Score tracking system
* "Next" button to navigate questions
* Restart quiz option
* Clean and responsive UI

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)

---

## 📂 Project Structure

* `index.html` → Structure of the quiz
* `quiz.css` → Styling and layout
* `quiz.js` → Quiz logic and functionality

---

## ▶️ How to Run the Project

1. Download or clone this repository
2. Open the project folder
3. Open `index.html` in your browser
4. Start answering the quiz

## 🎯 How It Works

* A question is displayed with multiple answer options
* Click on an answer to select it
* Correct answer is highlighted in green
* Incorrect answer is highlighted in red
* Click **Next** to move to the next question
* At the end, the total score is displayed
* Click **Play Again** to restart the quiz

---

## ⚙️ Core Logic

* Questions are stored in an array of objects
* JavaScript dynamically updates the question and answers
* Event listeners handle user interactions
* Score is updated based on correct answers
* DOM manipulation is used to update UI in real-time
